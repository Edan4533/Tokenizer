In this task i implmented tokenizer to king james bible

To run the code all needed is:
- download to your machine this repo 
- cd to this directory throw terminal
- run npm i
- run  : node app.js 

Contains 2 API : 
1 - click on this link "http://localhost:3000/tokenizeKingJamesBible" or copy and paste it in the browser;

2 - click on this link "http://localhost:3000/tokenizeByUrl/?url=URL" 
// URL variable provided by the user  
// example that i use: "http://localhost:3000/tokenizeByUrl/?url=https://www.w3.org/TR/PNG/iso_8859-1.txt"
